# Chicago Transit Authority (chicago-transit-authority)
The Chicago Transit Authority (CTA) provides Train Tracker and Bus Tracker APIs for developers to access real-time and schedule data for CTA train and bus service.

## APIs

### Chicago Transit Authority Train Tracker API
Train Tracker APIs provide real-time train arrival times and status for all CTA L train lines.
